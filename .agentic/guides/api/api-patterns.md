## A. No server, no endpoints — the package surface is the only API

This repository serves nothing: there are no HTTP handlers, no routes, and no
start command. The only "API" is the npm package's public surface — the default
`AiTriageReporter` export plus the re-exported types.
Evidence: `package.json` — `scripts` has `build`/`typecheck`/`lint`/`test`/`gate`/`release`, no `dev`/`start`; no server framework in any dependency block.
Evidence: `src/index.ts` — the sole entry point: `export default class AiTriageReporter implements Reporter` and `export type { AiTriageOptions, … } from './types.js'`.

**Rule: do not add HTTP endpoints, servers, or webhooks-as-listeners to this
package.** Outbound calls (Anthropic API, GitHub, Slack webhook) are covered by
`.agentic/guides/development/security-patterns.md`; changes to the public
package surface follow
`.agentic/guides/development/development-practices.md` §B–§C. If a feature
seems to require a served endpoint, escalate to a human — it contradicts the
self-hosted, nothing-to-run design in `README.md`.
